Instagram Comment Viewer

How to get the cURL command
  1. Open the Instagram post URL in your browser
  2. Right-click on the page and choose Inspect
  3. Open the Network tab (at the top of DevTools)
  4. Type graphql in the Network filter box
  5. Reload the page, then click the 🚫 icon to clear the network log
  6. Scroll down on the post until new comments auto-load
  7. A new API call will appear in the Network tab — right-click on it
  8. Choose Copy → Copy as cURL
  9. Paste the result below
